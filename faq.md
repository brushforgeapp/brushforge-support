# FAQ & Troubleshooting

**Restore Purchases**  
Open **Profile → Restore Purchases**. Make sure you’re signed into the same Apple ID used for the original purchase.

**Photos not saving**  
Check iOS **Settings → Privacy → Photos → BrushForge** and allow access.
